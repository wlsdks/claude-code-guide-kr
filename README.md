# 클로드 코드 기초 — 비개발자를 위한 인터랙티브 가이드

코드를 몰라도 **클로드 코드(Claude Code)**의 핵심 개념을 직접 눌러보며 이해할 수 있는 한 페이지 가이드입니다. 모든 개념을 버튼·애니메이션으로 시각화했어요.

## 🔗 바로 보기
배포되면 이 자리에 GitHub Pages 주소가 들어갑니다.

## 특징
- **파일 하나(`index.html`)** 로 된 정적 웹페이지 — 설치·서버 불필요, 더블클릭으로 바로 열림
- **완전 입문자(비개발자)** 대상 — 직접 안 써봐도 가이드만 보면 이해되도록 설계
- **다크모드 자동 대응 / 반응형**(모바일·노트북·대형 화면)

## 다루는 내용 (21개 주제)
- **시작하기** — 클로드 코드란?(챗 vs 코드 비교), 어디서 쓰나
- **기초 개념** — 컨텍스트 창, 세션 & 폴더, 토큰, 모델 고르기
- **기억과 규칙** — CLAUDE.md, 메모리, 스킬
- **능력 확장** — MCP(도구 연결), 플러그인, 서브에이전트, 이미지 붙여넣기, 자동화(훅)
- **안전하게** — 계획 모드, 권한 모드, 되돌리기, 데이터 & 프라이버시
- **실전** — 슬래시 명령, 요금제, 사용 한도, 용어 사전
